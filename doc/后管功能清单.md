## 二、字典 / 基础数据维护
- [ ] **语种字典维护(新增)** —— 平台维护「系统支持的语言」全集(语言编码 + 中英文名 + 排序 + 启停)。
  - 背景:参考系统 ByteTrack 的语种是读后端接口(`getAppLanguage` 返回 `{id, appId, language, type}`,type=1 默认/0 其他),按项目存表。
  - 现状(本期落地):项目级启用语种已建表 `mse_messenger_language`(project_id, language, type),租户在「常规设置 → 选择支持的语言」里增删/设默认;**可选语种全集(新增语言的候选项)暂用前端常量** `apps/console/src/constants/languages.ts`。
  - 后管要做:把"可选语种全集"从前端常量迁到**平台级语种字典表**(如 `pf_language`),后管可新增/编辑/停用语种;console 的语种候选从后端接口读取,实现"加新语种无需改代码"。
  - 迁移点:`constants/languages.ts` → 后端字典接口(单一来源,前端已集中,替换即可)。
